## Overview

This portfolio is intentionally minimal.

The goal is to keep the presentation clean and professional while still giving enough room to show technical work, current interests, and recent accomplishments.

## What Changed

This version introduces a markdown-based content system so projects and updates can be added more cleanly over time.

## Notes

The structure stays simple, but the content model is much easier to maintain and extend.
