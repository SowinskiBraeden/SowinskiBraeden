## Overview

XP-System was one of the projects that introduced me to the value of web applications and product thinking.

The core idea was to make school work feel more engaging by layering XP, levels, and a clearer sense of progress on top of classroom assignments.

## Focus Areas

- Building an interactive web application
- Thinking about user motivation and engagement
- Learning early lessons around application structure and workflows
