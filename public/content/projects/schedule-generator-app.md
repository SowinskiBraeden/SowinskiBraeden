## Overview

This desktop application was built to make my schedule generation algorithm easier to use and easier to test in a real workflow.

The project takes a difficult real-world scheduling problem and turns it into a usable tool for producing both student schedules and master timetables.

## Focus Areas

- Desktop tooling around a custom scheduling algorithm
- Parsing and transforming structured school data
- Exporting useful outputs for administrators and students

## Notes

This project is one of the clearest examples of my interest in backend logic, data processing, and software that solves practical operational problems.
